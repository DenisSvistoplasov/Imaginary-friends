# Imaginary Friends

Пример простого веб-приложения для создания и хранения воображаемых друзей.

Задеплоено на [GitHub Pages](https://denissvistoplasov.github.io/Imaginary-friends/).

## Для создания приложения использованы:
- SCSS
- Flex, Grid
- TypeScript
- Redux /Redux Thunk
- React Router v6+
- React Hooks
- Git
- Firebase
